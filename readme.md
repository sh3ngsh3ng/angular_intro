# Angular
- Framework used to build client side applications
- Modular Approach
    - Every feature is a separate module
    - Modules are reusable
- Quick & Easy development
- Unit testable
- Semantic Versioning
    - major.minor.patch
    - major change = break nay functionality
    - minor = added feature
    - patch = updates/bug fix
    - E.g. 7.2.11
